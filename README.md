# api-encomendas-projeto-rocketseat

## Tecnologias Usadas

- Typescript ( Types ) 
- Express ( API RESTFUL ) 
- PostgreSQL ( DataBase )
- Docker & Docker-Compose ( Container ) 
- Zod ( Validation )
- Prisma ( ORM )
- JWT ( Auth )
- Bcrypt ( Encrypt )
- Jest ( Unity Test )
